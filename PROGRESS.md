## Legend

- **Status (per block):** ✅ done · 🟡 partial · ⬜ pending · 🔴 missed / blocked
- **DSA (#):** status + number of problems solved that day.
- **Topic ✓:** the non-DSA block (Spring/system design/behavioral/applications). Use `—` when a day has no such block.
- **Time:** total focused minutes. **Conf:** 1–5 self-rating (1 = shaky, 5 = cold/whiteboard-ready).
- A day counts as **complete** for the streak when both its blocks are ✅ (or the day's only block is ✅).
