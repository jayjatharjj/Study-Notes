# 13-Week Interview Prep — Day-by-Day Checklists

Companion to [`career-plan-2026.md`](../../career%20planing/career-plan-2026.md) (market analysis + strategy) and [`../interview-qa.md`](../interview-qa.md) (Q&A bank).

**Goal:** walk into any interview — GCC, product, or services — able to answer *anything*, including curveballs. Each file has a daily checklist (DSA block + core topic block + self-check), the concepts to master with depth, sample questions including unexpected ones, an "extraordinary-candidate edge" section, and an end-of-week self-assessment gate.

**Cadence:** ~1.5 hr/weekday + ~4 hr each weekend day (~16 hrs/week). Never skip the daily DSA block. Weekday labels match the real 2026 calendar; Week 1 is a 5-day Wed–Sun kickoff so Week 2 onward is clean Mon–Sun.

### How to use
1. Each Monday, open that week's file and skim the goal + self-assessment gate first — that's your target.
2. Tick the daily checklist; log what you actually did.
3. Sunday: run the end-of-week self-assessment. **Don't advance if you fail a gate** — repeat the weak day instead. The schedule is a guide, not a deadline.
