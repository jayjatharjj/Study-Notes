# Week 6 — Apply Wave 1 (Aug 3–9, 2026)

> Study is done. This is the week you stop sharpening the blade and start swinging it. Profiles go live, the first applications go out in real batches, and the first referral asks land. The realistic failure mode now isn't "not good enough technically" — it's "prepped hard, sent twelve applications, waited for inbound." This week kills that failure mode with volume, referrals, and a tracker you update every single day. DSA drops to **maintenance** — 1–2 problems a day to stay warm — and the primary work becomes pipeline: applications, ATS-tailoring, referral outreach, and timed coding-round drills so the first OA doesn't catch you cold.

> 📨 **Apps & referrals this week:** 15–20 applications (target GCC/product + 8–12 safety-net) + 10–12 referral asks. See the [cadence & tracker](../applications-and-referrals.md).

---

## 🎯 Week Goal

1. Publish recruiter-optimised LinkedIn + Naukri + Instahyre + Cutshort profiles with **open-to-work and "available to join immediately" live**, so inbound starts filling while you sleep.
2. Submit **15–20 quality applications** in tracked batches — a curated **target GCC/product tier** plus a faster **mid-size/services safety-net** tier, run in parallel — every one ATS-tailored to the JD, every one logged.
3. Send **10–12 referral asks** to engineers (not HR) at target companies, each personalised, each logged.
4. Drill OA / timed-coding conditions until two mediums in 45 minutes feels routine — no hints, fresh editor, complexity stated.
5. Have a ready, non-anchoring answer to "available to join immediately?" and the "current/expected CTC" question before the first recruiter screen lands.

**Two-track strategy:** chase the **target (GCC/product)** roles *and* run the **floor/safety-net** track of faster-moving mid-size/services companies in parallel — from day one, not as a last resort. The safety net isn't a consolation prize; it's how you walk into the GCC loops already warmed up by real interview reps, and it protects cash flow while target loops mature. And **"available to join immediately"** is a genuine edge in a contested band — surface it everywhere: profiles, every application note, every referral ask, and the first screen the moment timelines come up.
