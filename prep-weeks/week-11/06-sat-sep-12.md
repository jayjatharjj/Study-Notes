# Week 11 · Day 6 — Sat Sep 12 — Confirm the Written Offer + Start BGV Docs

📌 **Today:** Get the decision in writing and remove every onboarding blocker. A verbal yes isn't an offer — the countersigned letter is.

**Primary activity:**

**Verify the written offer (45 min):**
- [ ] Confirm the final offer letter reflects everything agreed: base, variable, joining bonus (+ clawback terms), ESOP/LTI vesting, designation/band, start date, location/WFH.
- [ ] Anything verbal that isn't in the letter → get it added in writing before accepting. "We'll sort it later" is a red flag.
- [ ] Accept formally by email; request a countersigned copy.

**Start BGV / document prep (45 min):**
- [ ] Assemble one folder (digital + physical): prior offer/relieving/experience letters, last 3 salary slips, Form 16 / ITR, education certificates (10th/12th/degree + marksheets), PAN/Aadhaar, address proof.
- [ ] Since you've resigned, get your **relieving + experience letter** from your previous employer squared away now — it's the most common BGV blocker.
- [ ] Confirm the new company's BGV vendor + timeline so onboarding isn't delayed.

**Negotiate the start date (15 min):**
- [ ] Use immediate availability as a *choice*: offer an early start as goodwill, or take a short reset gap — your call. Confirm the date in writing.

**Pipeline:** Mark the chosen offer "accepted (pending letter)" → "accepted" once countersigned. Hold other live offers warm until the letter is in hand.

**EOD check:**
- [ ] Everything agreed is in the written letter? · [ ] BGV folder assembled? · [ ] Start date confirmed in writing?
