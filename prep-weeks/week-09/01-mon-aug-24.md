# Week 9 · Day 1 — Mon Aug 24 — Loop Entry: Tailoring + Light DSA

📌 **Today:** You are no longer preparing — you are IN the loops. Today is per-company tailoring: map each active JD to the project story you'll lead with, read one engineering-blog article per company, and get the pipeline sheet spotless before the live rounds start landing.

**DSA maintenance (~30 min):**
- [ ] One medium you haven't touched in 2+ weeks — **LC 79 (Word Search)**, **LC 200 (Number of Islands)**, or **LC 33 (Search in Rotated Sorted Array)**. Time it. Goal is to stay warm, not learn.
- [ ] Clean in ≤ 20 min? Done. If not, note the single gap (tree traversal? binary search edge?) and move on — no rabbit-holing.

**Primary activity:**

**Company tailoring block (45 min per active company):**
- [ ] Pull up each active JD. For each role:
  - Identify the **3 most-emphasised technical areas** (e.g., microservices, real-time pipelines, distributed systems).
  - Map each to a specific project story: **Smart360** (performance, security, AWS) vs **Deep Fathom** (Azure IaC, K8s, LLM async, CI/CD).
  - Note which story to **lead with** for this company — don't lead the same story every time.
  - Google "[Company] tech blog", "[Company] engineering stack", "[Company] recent launch". Read one article. Extract one real observation.
  - Check LinkedIn for known interviewers — shared tech = shared language.

**Pipeline sheet hygiene (15 min):**
- [ ] Every row has: Company | Role | JD URL | Stage | Last Action | Next Action | Next Action Date | Comp Range | Interviewer(s) | Notes.
- [ ] Update stale rows. Waiting > 5 business days on a company? Send a polite follow-up today: "Following up on my [round] on [date] — happy to provide any additional context."

**Pipeline:** This is the week the sheet earns its keep. Log every interaction within 24 hours. Every row needs a Next Action + Date — a row without one is a dead lead. Surface immediate availability when start dates come up — "available to join immediately."

**EOD check:**
- [ ] For each active company: can you name their primary product, likely stack, and one specific thing you'd bring that you haven't mentioned before?
- [ ] Does each active company have a designated lead-story and a real observation noted?
- [ ] Pipeline sheet has zero stale rows?
- [ ] DSA warm-up done?
